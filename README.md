# Tic---Tac---Toe-
Tic- Tac- Toe  - This is a browser based tic tac toe game built with HTML , CSS and JAVASCRIPT.

## features :-
- two players can play
- winner  and draw detection
- have reset and start game button
- responsive  and interactive  design
-

 I built this as a practice purpose to improve my skills and JavaScript fundamentals like DOM manipulation . event handling and game logic implementation.
